## 🔧 Technical Stack

**Frontend:**
- Tailwind CSS 3.x (utility-first)
- Alpine.js 3.x (reactivity)
- HTMX (dynamic updates)
- Font Awesome 6.5 (icons)
- Inter Font (Google Fonts)

**Backend:**
- Go 1.22 + Fiber
- PostgreSQL + GORM
- HTML Templates
